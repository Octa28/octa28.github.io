WORD – FORMATARE TEXT

Culoare text

Selectezi text → Acasă → A colorat (săgeata) → alege culoarea (Font Color).

Bold / Italic / Subliniere

Ctrl+B=Aldin(Bold) | Ctrl+I=Cursiv(Italic) | Ctrl+U=Subliniere(Underline)

Subliniere ondulată colorată

Selectezi → Acasă → săgeata lângă S → Mai multe sublinieri → Stil: ondulat(~) → Culoare subliniere: verde/altă culoare → OK

Tăiat / Strikethrough

Acasă → butonul abc tăiat. Sau Format Font → Efecte → Tăiat.

Superscript / Subscript

Ctrl+Shift++=superscript | Ctrl+==subscript. Sau Acasă → x² / x₂

Spațiere între caractere (Extended)

Selectezi → Acasă → săgeata mică Font → fila Avansat → Spațiere: Extinsă → valoare (ex: 5pt) → OK

Dimensiune / Tip font

Acasă → câmpul cu numărul (ex:12) sau cu fontul (ex:Arial) → tastezi → Enter. Sau Ctrl+Shift+P pt dimensiune.

Majuscule / UPPERCASE

Selectezi → Acasă → butonul Aa. Sau Shift+F3 ciclează.

Pensulă formatare / Format Painter

Click pe text sursă → Acasă → Pensulă (pictograma pensulă) → selectezi destinație. Dublu-click pe pensulă = aplici de mai multe ori.

Ștergere formatare

Selectezi → Acasă → A cu radieră sau Ctrl+Space.

WORD – FORMATARE PARAGRAF

Aliniere text

Ctrl+L=Stânga | Ctrl+R=Dreapta | Ctrl+E=Centru | Ctrl+J=Justify

Coloane / Columns

Selectezi textul → Aspect → Coloane → Mai multe coloane → nr, spațiere, Linie separatoare → Aplicare la: Text selectat → OK. Selectează textul ÎNAINTE!

Indentare / Indent

Click în paragraf → Acasă → săgeata Paragraf → Indentare Stânga/Dreapta (ex: 2,54cm=1"). Sau trage rigla.

Spațiere rânduri / Line Spacing

Ctrl+1=simplu | Ctrl+2=dublu | Ctrl+5=1,5. Sau Acasă → Spațiere rânduri.

Spațiere înainte/după paragraf

Aspect → câmpurile Înainte/După din grupul Paragraf. Sau dialog Paragraf → Spațiere.

Marcatori personalizați (ex: @)

Selectezi → Acasă → săgeata lângă Marcatori → Definire marcator nou → Simbol → găsești @ → OK

Numerotare / Numbering

Acasă → Numerotare. Click dreapta pe numerotare → Setare valoare numerotare pt a schimba startul.

Sortare alfabetică paragrafe

Selectezi paragrafele → Acasă → AZ↓ (Sort) → Ascendent → OK

Borduri paragraf (stiluri diferite pe laturi)

Selectezi paragraful → Acasă → Borduri (săgeata) → Borduri și umbrire → Particularizat → alegi stil1 → click latură stângă în preview → alegi stil2 → click latură dreaptă → Aplicare la: Paragraf → OK

Umbrire paragraf / Shading

Acasă → Borduri și umbrire → fila Umbrire → culoare fundal → Aplicare la: Paragraf.

Stiluri paragraf

Acasă → Stiluri (galeria): Titlu 1, Titlu 2, Normal etc.

Drop Cap

Click în paragraf → Inserare → Drop Cap → Înfășurat/În marjă.

WORD – IMAGINI

Inserare imagine

Inserare → Imagini → din fișier → navighezi → Insert.

Resize exact

Click pe imagine → Format imagine → câmpurile Înălțime/Lățime (dreapta sus). Debifează Blocare proporții dacă vrei valori independente.

Flip vertical

Click imagine → Format imagine → Rotire → Răsturnare verticală (Flip Vertical).

Flip orizontal

Format imagine → Rotire → Răsturnare orizontală (Flip Horizontal).

Contrast / Luminozitate

Click imagine → Format imagine → Corecții (Corrections) → Opțiuni corecții imagine → cursorul Contrast la valoarea cerută (ex: 80%). LibreOffice: click dreapta → Format imagine → fila Imagine.

Transparență imagine

Format imagine → Imagine → Transparență (câmpul %).

Wrap text

Format imagine → Încadrare text → În pătrat / Aproape / Prin / Deasupra și dedesubt / Niciunul.

Legendă / Caption

Click imagine → Referințe → Inserare legendă → Etichetă: Imagine → Poziție: Deasupra → Numerotare: A,B,C → OK. Repetă pentru fiecare imagine.

WORD – TABELE

Inserare tabel

Inserare → Tabel → trage în grilă SAU Inserare tabel → specifici rânduri și coloane → OK.

Înălțime rând exact (ex: 3cm=1.18")

Selectezi rândurile → fila Aspect [Instrumente tabel] → câmpul Înălțime celulă → introduci valoarea → Enter. 1"=2,54cm | 3cm=1,18" | 2,54cm=1"

Lățime coloană exactă

Fila Aspect → câmpul Lățime celulă. Sau trage marginea coloanei.

Aliniere conținut în celulă

Selectezi celule → Aspect → grup Aliniere → alege una din 9 butoane.

Fuzionare celule / Merge

Selectezi celulele → Aspect → Unire celule (Merge Cells).

Scindare celulă / Split Cells

Click în celulă → Aspect → Scindare celule → nr coloane și rânduri → OK.

Ștergere conținut celule (nu celulele)

Selectezi celulele → Delete. Nu Backspace (șterge celule)!

Borduri tabel: exterior dublu, interior punctat

Selectezi tot tabelul → Acasă → Borduri → Borduri și umbrire → Particularizat → stil=dublu → click 4 laturi externe → stil=punctat → click linii interioare → OK.

Eliminare borduri celule

Selectezi celulele → Borduri și umbrire → Particularizat → click laturile de eliminat din preview → OK. Sau: Fără borduri.

Inserare rând/coloană

Click în tabel → Aspect → Inserare deasupra/dedesubt/stânga/dreapta.

Ștergere rând/coloană

Selectezi rândul/coloana → Aspect → Ștergere → rânduri/coloane.

Orientare pagină

Notă de subsol / Footnote

Protejare document

Revizuire → Restricționare editare → alege tipul → Aplicare protecție → parolă → OK.

Track Changes

Simbol special

WORD – GĂSIRE / ÎNLOCUIRE / CORECTARE

Găsire

Ctrl+F → tastezi textul → Enter.

Înlocuire (ex: "un" → "*")

Ctrl+H → Găsiți: un → Înlocuiți cu: * → Înlocuiți tot → notezi nr. înlocuiri din popup → OK. Bifează "Potrivire majuscule" dacă se cere litere mici!

Ortografie și gramatică

F7 sau Revizuire → Ortografie și gramatică. Acceptă/ignoră sugestiile.

Autocorectare

Fișier → Opțiuni → Verificare → Opțiuni autocorectare → adaugi reguli personalizate.

Corectare ortografică în timp real

Linii roșii = erori ortografice. Linii verzi/albastre = erori gramaticale. Click dreapta pe cuvânt → sugestii.

HARDWARE – COMPONENTE

CPU (Unitatea Centrală de Procesare)

"Creierul" calculatorului. Caracterizat de: frecvență (GHz) – viteza de calcul; număr nuclee (cores). Ex: Intel Core i5 3.2GHz, AMD Ryzen 5. Mai mult GHz și mai multe nuclee = performanță mai bună.

RAM (Random Access Memory)

Volatilă (se șterge la oprire). Stochează datele programelor în execuție. Unitate: GB. Ex: 8GB, 16GB DDR4. Mai mult RAM = mai multe programe simultane, sistem mai fluid.

ROM (Read Only Memory)

Nevolatilă (păstrează datele la oprire). Conține firmware-ul (BIOS). Nu poate fi modificată de utilizator în mod normal.

BIOS / UEFI

Firmware pe placa de bază. Inițializează hardware-ul la pornire, verifică componentele (POST), lansează SO. Acces la pornire cu Del, F2 sau F12.

INTERNET – CONCEPTE

Internet vs WWW

Internet=infrastructura fizică globală (rețele, cabluri, routere, servere). WWW=serviciu pe internet, pagini web interconectate prin hyperlink-uri. WWW este parte din internet, nu același lucru.

Adresa IP

Adresa numerică unică a unui dispozitiv. IPv4: 4 grupuri 0-255 (ex: 192.168.1.1). IPv6: format hexazecimal mai lung. IP static=fix | IP dinamic=se schimbă la fiecare conexiune.

DNS (Domain Name System)

Convertește un nume de domeniu (google.com) în adresă IP. Google DNS: 8.8.8.8 | Cloudflare: 1.1.1.1.

URL (Uniform Resource Locator)

Structură: protocol://domeniu:port/cale?parametri. Ex: https://www.google.com/search?q=bac.

Domenii de nivel superior (TLD)

.ro=România | .com=comercial | .org=organizație | .net=rețea | .gov=guvern | .edu=educație.

Server și Client

Server=calculator care oferă servicii (web, email, fișiere), rulează 24/7. Client=dispozitivul care accesează serviciile (browser-ul tău). Host=orice dispozitiv cu adresă IP în rețea.

SERVICII INTERNET

VoIP (Voice over Internet Protocol)

Telefonie prin internet. Vocea digitizată se transmite ca pachete de date. Exemple: Skype, Zoom, WhatsApp, Viber, Google Meet, Teams. Avantaj: cost redus/gratuit, video conferință.

E-commerce (Comerț electronic)

Pași: căutare → coș → cont → adresă livrare → plată (card/PayPal/ramburs) → confirmare. Avantaje: 24/7, prețuri mai mici, comparare ușoară. Dezavantaje: nu testezi produsul, risc fraude. Exemple: eMag, Amazon.

E-banking (Banking online)

Verificare sold, transferuri, plăți facturi, extras de cont online. Necesită autentificare 2FA (parolă + SMS).

Cloud computing

Stocare și procesare date pe servere la distanță. Exemple: Google Drive, OneDrive, Dropbox, iCloud. Avantaje: acces de oriunde, backup auto. Dezavantaje: nevoie de internet, confidențialitate.

Chat / Mesagerie instant

Comunicare în timp real text/voce/video. WhatsApp, Telegram, Discord, Messenger, Signal.

Streaming

Reddit, Stack Overflow, forumuri tematice. Mesaje publice organizate pe thread-uri.

SECURITATE ÎN INTERNET

Phishing

Email/site fals care imită o entitate legitimă. Semne: adresă expeditor greșită, linkuri ciudate, urgență falsă, greșeli gramaticale. Nu introduci date pe linkuri din email!

Parole puternice

Min 12 caractere: MAJUSCULE + minuscule + cifre + simboluri. Parolă diferită pentru fiecare cont. Nu folosi date personale (nume, data nașterii).

2FA/MFA (Autentificare cu doi factori)

Parolă + cod SMS/email/aplicație. Chiar dacă parola e furată, contul rămâne protejat.

Netiquette online

Respectă intimitatea. Nu distribui date personale ale altora. Nu hărțuiești (cyberbullying). Verifică informațiile înainte de a le distribui (fake news). Respectă drepturile de autor. Limbaj decent.

Semnătură digitală

Verifică identitatea semnatarului și integritatea documentului electronic. Valoare juridică. Bazată pe PKI (cheie publică/privată).
